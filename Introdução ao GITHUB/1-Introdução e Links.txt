Link para download do GIT
https://git-scm.com/downloads

Comandos principais:
git init - iniciar o git
git add - iniciar o versionamento
git commit -m - criar um commit
git clone - clonar um código do github para usar na máquina
git config
git push origin main - Enviar o arquivo atualizado para o GitHub
git status

mkdir - criar pasta
cd - entrar na pasta ou arquivo
cd .. - voltar uma pasta
ls - listar as pastas e arquivos
mv arquivo.md ./pasta/ - mover um arquivo para uma pasta
